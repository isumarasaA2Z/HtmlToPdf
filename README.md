# HTML to PDF Conversion Service

An ASP.NET Core 8.0 Web API that converts HTML templates with dynamic data to PDF documents using PuppeteerSharp (headless Chrome).

## Features

- Convert HTML templates to PDF with dynamic text and table replacement
- Support for custom page setup (size, orientation, margins)
- Configurable headers and footers with page numbering
- Template system for reusable HTML layouts
- RESTful API with Swagger documentation
- Comprehensive error handling and logging

## Prerequisites

- .NET 8.0 SDK
- Windows, Linux, or macOS
